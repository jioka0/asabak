HOW THE WEBSITE WORKS
=====================

1. LOADING PROCESS
   - Page loads with preloader animation
   - imagesLoaded waits for all images to load
   - Preloader fades out, revealing content
   - GSAP animations initialize

2. USER INTERACTIONS
   - Scroll: Triggers parallax, animations, and scrollspy updates
   - Click navigation: Smooth scrolls to sections
   - Click theme switcher: Toggles between light/dark themes
   - Click project images: Opens PhotoSwipe gallery
   - Click social links: Opens external social media
   - Submit contact form: AJAX submission with success message
   - Click CTA buttons: External redirects

3. RESPONSIVE BEHAVIOR
   - Layout adapts to screen size
   - Navigation becomes mobile-friendly
   - Typography scales appropriately
   - Images and spacing adjust

4. THEME SYSTEM
   - Detects system preference on load
   - Stores user choice in localStorage
   - Dynamically updates CSS variables
   - Smooth transitions between themes
